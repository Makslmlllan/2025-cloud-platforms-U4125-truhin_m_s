## Теоретическая часть

Облачные хранилища предоставляют возможность хранения данных в распределенной инфраструктуре.

Существуют три основные модели хранения:

* **Блочное хранилище (Block Storage)** — используется для работы с виртуальными машинами
* **Файловое хранилище (File Storage)** — иерархическая структура файлов
* **Объектное хранилище (Object Storage)** — хранение данных в виде объектов

В рамках данной лабораторной работы использовалось **объектное хранилище (Cloud Storage)**, которое подходит для хранения файлов, изображений и медиа-контента.
